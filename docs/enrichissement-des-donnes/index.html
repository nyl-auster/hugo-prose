Enrichissement des données du site

March 12, 2017 by Yann

opendata

Le site existera depuis une petite semaine d'ici demain, nous avons déjà néanmoins réussi à effectuer un gros travail de traitement sur les données et pouvons mettre à disposition tout plein de JSON dont l'exploitation permettra de générer des synthèses parlantes sur l'équité de temps de parole d'antenne à la présidentielle 2017.

Si vous êtes développeuse / développeur, nous serions ravis de vous voir utiliser nos données pour leur donner vie en utilisant les librairies de votre choix. Et si vous le faites, contactez-nous pour nous montrer vos oeuvres !

Nous allons aussi travailler à mettre à dispostion une version CSV de ces synthèses en JSON au cours de la semaine.
