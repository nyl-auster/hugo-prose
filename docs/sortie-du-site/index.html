Sortie du site

March 7, 2017 by Yann

Aujourd'hui 7 mars, après une soirée de quelques heures de codage, nous livrons la première release de notre projet. Il s'agit donc d'une version alpha mais qui propose déjà un premier JSON exploitable par toutes et tous pour créer votre propre synthèses des temps d'antenne des candidats à la présidentielle 2017.

Il y aura rapidement dans les jours à venir de nouveaux JSON permettant d'avoir par exemple le temps de parole des candidats par chaînes de télévision.
